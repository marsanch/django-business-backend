# WhatsApp Appointment Reminder

Sistema de recordatorios automáticos de citas por WhatsApp.

## Tecnologías
Python
Django
API WhatsApp

## Funcionalidades
- Consulta citas pendientes
- Envía recordatorios automáticos
- Evita envíos duplicados

## Uso
Pensado para negocios con citas.
